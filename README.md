🛍️ Amazon Clone – Frontend Project
A responsive, feature-rich Amazon-like shopping experience built using HTML, CSS, and JavaScript. This project simulates a full front-end e-commerce flow including cart management, order tracking, and basic checkout functionality.

🚀 Features
🛒 Add to Cart – Add products with quantity control and price calculations

🧮 Real-time Cart Updates – Totals auto-update using JavaScript + localStorage

📦 Order Tracking – Simulated past orders with product thumbnails

🧱 Modular Code Structure – Separated concerns using JS modules

🧠 Object-Oriented Programming – Implemented using both Object Syntax and ES6 Classes

✅ Unit Testing – Jasmine test suite for key cart logic

🌐 Responsive UI – Clean, scalable design using grid layouts

💡 Technologies Used
HTML5, CSS3 (Grid/Flexbox)

Vanilla JavaScript (ES6+)

Jasmine (for testing)

LocalStorage API
